Data
The high-fidelity UTB benchmark data are publicly available from the Geothermal Data Repository hosted by Lawrence Berkeley National Laboratory:

Nico, P., Zhang, Y., Liu, X., & Doughty, C. (2022).
Community Resilience through Low-Temperature Geothermal Reservoir Thermal Energy Storage.
Geothermal Data Repository.
DOI: 10.15121/1843040

The original benchmark data are not redistributed in this repository. Users should download the dataset from the official repository and update the file paths in the notebook or scripts.
